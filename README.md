# 시간표 랜덤 추천기

개설된 강의에 한하여 시간표를 가져와, 사용자가 제시한 조건에 맞는 시간표를 랜덤으로 생성합니다.
그렇게 된다면 추후 시간표를 맞춰볼 필요없이 자동으로 만들어진 것에서 골라서 수강신청을하면 된다.
특히 해당과목의 수강신청이 실패했을 경우에 추천하는 대체과목도 알려주도록 설정한다.





## ========= 설계 ==========

프로젝트 이름: 시간표 추천 시스템 (Timetable Recommendation System)

## 🎯 목표
사용자가 선택한 개설 과목들만을 기반으로 가능한 모든 시간표 조합을 생성하고,
그 중 조건에 맞는 최적의 시간표 5개를 추천하는 시스템을 만든다.

랜덤이 아니라 "사용자 선택 기반 조합 + 제약조건 + 점수 기반 추천" 구조이다.

---

## 📚 핵심 기능

### 1. 개설과목 조회 화면
- 과목 목록 표시
- 필터 기능 포함:
  - 학과 (자동완성 지원: 예: "컴공" → "컴퓨터공학과", "행정" → "행정학과")
  - 과목 구분 (전공필수 / 전공선택 / 교양필수 / 교양선택)
  - 학점 (1~6학점)
- 사용자가 체크해서 과목 선택 가능

---

### 2. 시간표 생성 조건
- 사용자가 선택한 과목만 사용해서 시간표 생성
- 선택된 과목들로 가능한 모든 조합 생성 (백트래킹/DFS 가능)
- 시간 충돌이 발생하면 제외

---

### 3. 시간표 제약 조건
- 최대 학점: 목표 학점 (예: 18)
- 허용 범위: 목표 -1 ~ 목표 (예: 17~18학점 허용)
- 학점 초과는 절대 불가
- 1교시 포함 여부에 따라 감점 또는 제외 (전략 선택에 따라 다름)

---

### 4. 전략 선택 기능 (상단 UI)
사용자는 아래 전략 중 하나 선택:

- 금공강 우선
- 월공강 우선
- 최대학점 수강
- 1교시 최소화
- 1교시 배척
- 전공필수 우선
- 교양필수 우선

각 전략에 따라 점수 계산 방식이 달라진다.

---

### 5. 추천 시스템
- 가능한 시간표 생성 후 점수 계산
- 점수 기준:
  - 공강 많음
  - 연강 적음
  - 1교시 없음
  - 전략 조건 충족 여부
- 상위 5개 시간표 추천

---

### 6. UI 구조 (Swing)

레이아웃:

(1) 상단 패널
- 전략 선택 JComboBox
- "시간표 생성" 버튼

(2) 좌측 패널
- 개설과목 필터 UI
- 학과 자동완성 입력창
- 과목 구분 ComboBox
- 학점 필터
- 과목 체크 리스트

(3) 우측 패널
- 추천 시간표 5개 JList
- 각 시간표 점수 표시

(4) 하단 패널
- 선택된 시간표 상세 JTable
- 과목 상세 정보 표시
- 대체 과목 추천 리스트

---

### 7. 상세 기능

#### 시간표 클릭 시
- 해당 시간표의 과목 목록 표시
- JTable로 시간표 시각화

#### 과목 클릭 시
- 교수, 학점, 강의실 정보 출력

#### 수강 실패 시
- 유사 과목 추천 기능 제공
  - 같은 학과
  - 같은 과목 구분
  - 유사 키워드 기반

---

### 8. 데이터 구조

Course
- name
- professor
- department
- type (전필/전선/교필/교선)
- credit
- timeSlot

Schedule
- List<Course>
- totalCredit
- score

RecommendationEngine
- 시간표 생성
- 점수 계산
- 필터링

---

### 9. 구현 요구사항
- Java Swing 기반 GUI
- 객체지향 설계 필수
- 백트래킹 또는 DFS로 시간표 생성
- 시간 충돌 검사 로직 포함
- 전략 기반 점수 시스템 구현

---

## 🚀 추가 요구 (중요)
- 랜덤 생성이 아니라 "선택된 과목 기반 생성"
- 모든 시간표는 유효한 시간 충돌 없는 구조여야 함
- 추천 이유를 점수 breakdown으로 표시할 것

---

## 📌 출력 결과
- Java 전체 코드 (가능하면 클래스 분리)
- Swing UI 포함
- 추천 엔진 포함
- 실행 가능한 구조
